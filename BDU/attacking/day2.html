Day 2: Movement & Runs

Getting open and creating space

Drill 1: Checking To and Away

10 minutes Pairs Beginner

Practice the check-away-check-to movement. Start deep, check toward ball, then spin and run into space. Timing is everything.

  • Check away first to create space
  • Sharp change of direction
  • Show for the ball with hand signal
  • Receive on back foot, open to field

Drill 2: Diagonal Runs

15 minutes Small Group Intermediate

Diagonal runs behind the defense. Start wide, run diagonally toward goal. Stay onside, time your run with the pass.

  • Start from wide position
  • Bend your run to stay onside
  • Eye contact with passer before run
  • Sprint at full speed once you commit

Drill 3: Overlapping Runs

15 minutes Pairs/Trios Intermediate

Overlap your teammate to create 2v1 situations. Runner goes around the outside, ball carrier can pass or use the space created.

  • Call "OVERLAP!" so teammate knows
  • Sprint past at full speed
  • Stay wide to stretch defense
  • Be ready for cross or cut-back

Drill 4: Movement Game

20 minutes 5v5 Advanced

5v5 game where goals only count if scorer made a run before receiving. Forces constant movement off the ball.

  • Never stand still - always be moving
  • Create space for yourself AND teammates
  • Communicate your runs
  • Time runs to stay onside

Key Coaching Points

  • Timing: A well-timed run beats a fast run
  • Communication: Eye contact and calls with passer
  • Selfless Running: Runs can create space for others
  • Stay Onside: No goal if you're offside - patience!
  • Never Stop: Movement creates opportunities
