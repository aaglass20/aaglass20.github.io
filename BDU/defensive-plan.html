Defensive Training Plan

About This Plan

This defensive training plan focuses on building solid defensive fundamentals through progressive daily sessions. Each day builds upon the previous, covering topics like positioning, 1v1 defending, team shape, pressing, and recovery runs. Complete each day's drills before moving to the next.

Training Days

Day 1 Defensive Stance & Positioning Day 2 1v1 Defending Day 3 Team Shape Day 4 Pressing & Recovery

Add New Day

To add a new training day:

  1. Create a new HTML file in the defensive/ folder (e.g., day5.html)
  2. Copy the structure from an existing day file
  3. Add the drills for that day
  4. Add a new card link above in the day-grid section
